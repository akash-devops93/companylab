# Linux Basic Practice Lab

This repository contains my hands-on Linux practice performed on RHEL.

## Topics Covered
- Linux basic commands
- File and directory management
- Hidden files
- Permissions using chmod
- Disk usage commands
- Filesystem checking
- Navigation commands
- File creation and deletion
- Mount and storage basics
- Command history

## Commands Practiced
- pwd
- ls
- ls -l
- ls -a
- mkdir
- cd
- touch
- chmod
- df
- du
- mount
- lsblk
- rm
- history

- RHEL 10
